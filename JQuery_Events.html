JQuery Events

jQuery está hecho a medida para responder a eventos en una página HTML.
¿Qué son los eventos? Todas las acciones de los diferentes visitantes a las que puede responder una página web se denominan eventos. Un evento representa el momento preciso en que sucede algo. Ejemplos: mover el mouse sobre un elemento seleccionando un botón de radio haciendo clic en un elemento El término "incendios / despedidos" se usa a menudo con eventos. Ejemplo: "El evento de pulsación de tecla se activa en el momento en que pulsa una tecla". A continuación, se muestran algunos eventos DOM habituales:
Mouse Events Keyboard Events Form Events Document/Window Events
click keypress submit load
dblclick keydown change resize
mouseenter keyup focus scroll
mouseleave   blur unload
