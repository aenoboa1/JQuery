Selectores jQuery

Los selectores jQuery son una de las partes más importantes de la libreria jQuery.

¿Para qué sirven los selectores jQuery?

Los selectores jQuery permiten seleccionar y manipular elementos HTML.
Los selectores jQuery son usados para "encontrar" (o seleccionar) elementos HTML basados en su nombre, id, clases, typos, atributos, valor de los atributos y mucho más. Está basado en Selectores CSS, y adicionalmente, tiene sus propios selectores personalizados.
Todos los selectores en jQuery comienzan con el signo de dolar y el paréntesis: $().

El selector de elementos

El selector de elementos jQuery selecciona elementos basándose en el nombre del elemento.
Se pueden seleccionar todos los elementos <p> en una página de la siguiente manera:
$("p")

Ejemplo

Cuando el usuario hace clic en un botón, todos los elementos <p>

El selector #id

El selector #id de jQuey usa el atributo id de una etiqueta HTML para encontrar el elemento especifico.
Un id debería ser único dentro de una página, así que se debería usar el selector #id cuando se encontrar un solo y único elemento.
Para encontrar elementos con un id específico, escribir un caracter hash, seguido del id del elemento HTML:
$("#test")

Ejemplo

Cuando el usuario hace clic al botón, el elemento con id="#test"

El selector .class

El selector .class de jQuery encuentra elementos de una clase específica.
Para encontrar elementos con una clase específica, se escribe el caracter de periodo, seguido del nombre de la clase:
$(".test")
